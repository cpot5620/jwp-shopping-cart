  - [x] 사용자를 선택하면, 이후 요청에 선택한 사용자의 인증 정보가 포함된다.
  
  - [x] 요청 헤더의 Authorization 필드를 사용해 사용자 인증 처리를 한다.

  - [ ] 장바구니에 담긴 상품을 제거할 수 있다.
  - [x] "/cart" uri로 접근해 장바구니 목록을 조회할 수 있다.
