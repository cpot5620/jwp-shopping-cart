### API

#### 생성

|     URL     |                  /productEntities                  |
|:---:|:-------------------------------------------:|
| HTTP METHOD |                    POST                     |
 | BODY | String: Name,  int: Price, String: ImageUrl |


#### 수정

|     URL     |                      /productEntities                       |
|:---:|:----------------------------------------------------:|
| HTTP METHOD |                         PUT                          |
| BODY | int: Id, String: Name,  int: Price, String: ImageUrl |

#### 삭제

|     URL     | /productEntities |
|:---:|:---------:|
| HTTP METHOD |  DELETE   |
| BODY |  int: Id  |
